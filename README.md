# SGF.work Public Roadmap

Welcome! This is the public home for **feature requests, bug reports, discussions, and the product roadmap** for [SGF.work](https://sgf.work).

The application source code lives in a separate private repository. This repo exists so that anyone — users, customers, contributors, and the curious — can see what we're working on, weigh in on what to build next, and report problems.

---

## What belongs here

- **Feature requests** — ideas for new functionality
- **Bug reports** — things that are broken or behaving unexpectedly
- **Roadmap discussion** — feedback on planned work and priorities
- **Questions & discussions** — open-ended conversations about the product

## What does *not* belong here

- Source code, pull requests against application code, or implementation details
- Security vulnerabilities — please report these privately (see below)
- Account-specific support requests containing personal data

---

## How to file something

### 🐛 Bug reports
Open a new issue using the **Bug Report** template. Please include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Browser / OS / device (if relevant)
- Screenshots or screen recordings when possible

### ✨ Feature requests
Open a new issue using the **Feature Request** template. Please include:
- The problem you're trying to solve (not just the proposed solution)
- Who else this would help
- Any examples from other products you've seen do this well

### 💬 Discussions
For open-ended conversation, head to the **Discussions** tab rather than opening an issue.

---

## Roadmap

The live roadmap is tracked in the **Projects** tab. Items generally move through these stages:

| Stage | Meaning |
|---|---|
| `triage` | Newly filed, not yet reviewed |
| `under consideration` | We've read it and are thinking about it |
| `planned` | Scheduled for a future release |
| `in progress` | Actively being worked on |
| `shipped` | Released — see the linked version |
| `declined` | We've decided not to do this (with reasoning) |

A label of `declined` is not a judgment on the idea — it usually means the idea doesn't fit our current direction or scope.

---

## Voting & prioritization

We use 👍 reactions on the **first comment** of an issue as the primary signal of demand. Please react instead of leaving "+1" comments — it keeps threads readable and lets us sort by popularity.

---

## Security

**Do not file security vulnerabilities as public issues.** Email security@sgf.work (or use GitHub's private vulnerability reporting on this repo) so we can address the issue before it becomes public.

---

## Code of conduct

Be kind, be specific, and assume good faith. Disrespectful or harassing behavior will result in removal from the repo. We reserve the right to close, lock, or delete issues and comments that don't follow these norms.

---

## Cross-references

Internal commits and pull requests in our private code repository may reference issues here using `GlobalWorkspaceHoldings/roadmap#123`. When work ships, the corresponding issue will be closed and labeled `shipped` with a link to the release notes.
